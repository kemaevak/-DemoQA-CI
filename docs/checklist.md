# Чек-листы

Чек-листы составлены по результатам исследовательского тестирования
и легли в основу тест-кейсов ([test-cases.md](test-cases.md)).

Обозначения: ✅ — проверка пройдена, ❌ — найден дефект (номер в скобках).

## Elements / Text Box

- ✅ Форма отправляется с корректными данными во всех полях
- ✅ Введённые данные отображаются в блоке результата без искажений
- ✅ Кириллица в полях обрабатывается корректно
- ❌ Email некорректного формата: поле подсвечивается, но текста ошибки нет (BUG-009)
- ❌ Форма отправляется полностью пустой — нет обязательных полей (BUG-002)

## Elements / Check Box

- ✅ Узлы дерева чекбоксов разворачиваются и сворачиваются
- ✅ Выбор корневого элемента отмечает все вложенные
- ✅ Выбор одного элемента не затрагивает соседние
- ✅ Список выбранных элементов соответствует отмеченным чекбоксам
- ✅ Повторный клик снимает выбор

## Elements / Radio Button

- ✅ Выбор Yes отображается в результате
- ✅ Выбор Impressive отображается в результате
- ✅ Радиокнопки взаимоисключающие
- ❌ Вариант No недоступен для выбора (BUG-001)

## Elements / Web Tables

- ✅ Добавление записи через форму регистрации
- ✅ Поиск фильтрует таблицу по всем колонкам
- ✅ Редактирование записи сохраняет изменения
- ✅ Удаление записи убирает её из таблицы
- ✅ Пустая форма регистрации не отправляется, поля подсвечиваются
- ✅ Email в форме регистрации проверяется на формат
- ❌ Принимаются бессмысленные значения: возраст 0, зарплата 0 (BUG-007)

## Elements / Buttons

- ✅ Двойной клик выводит сообщение
- ✅ Клик правой кнопкой выводит сообщение
- ✅ Обычный клик по кнопке с динамическим id выводит сообщение
- ✅ Сообщения не появляются при неправильном типе клика

## Elements / Links

- ✅ Simple link и Dynamic link открывают главную страницу в новой вкладке
- ✅ API-ссылки выводят статус ответа
- ❌ В сообщении о статусе опечатка: «staus» вместо «status» (BUG-003)

## Elements / Upload and Download

- ✅ Файл загружается, имя отображается на странице
- ✅ Кнопка Download скачивает файл sampleFile.jpeg
- ✅ Загружается файл с кириллицей в имени

## Elements / Dynamic Properties

- ✅ Кнопка становится активной через 5 секунд
- ✅ Кнопка меняет цвет текста
- ✅ Кнопка появляется через 5 секунд

## Forms / Practice Form

- ✅ Форма отправляется со всеми заполненными полями
- ✅ Форма отправляется только с обязательными полями (имя, фамилия, пол, телефон)

- ✅ Пустая форма не отправляется
- ✅ Данные в модальном окне результата соответствуют введённым
- ❌ В поле Select Picture загружается файл любого типа, не только изображение (BUG-006)

## Alerts, Frame & Windows

- ✅ New Tab открывает новую вкладку с образцом страницы
- ✅ Alert открывается и закрывается
- ✅ Alert с задержкой появляется через 5 секунд

- ✅ Prompt: введённый текст отображается в результате
- ✅ Содержимое iframe читается, вложенные фреймы работают
- ✅ Маленькое и большое модальные окна открываются и закрываются

## Widgets

- ✅ Accordion: первая секция открыта по умолчанию, клики переключают секции
- ✅ Select Menu: значения выбираются в классическом и кастомном селектах
- ✅ Progress Bar: запускается, останавливается, сбрасывается после 100%
- ✅ Slider: ползунок двигается мышью и клавиатурой, значение обновляется
- ✅ Date Picker: дата выбирается через календарь
- ✅ Tool Tips: подсказка появляется при наведении
